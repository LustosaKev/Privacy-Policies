Política de Privacidade – Demo do Jogo

Última atualização: 22/11/2025

Esta Política de Privacidade descreve como o aplicativo Demo do Jogo lida com informações do usuário. O objetivo é garantir transparência e conformidade com as regras da Google Play.

1. Coleta de Dados

O aplicativo não coleta dados pessoais de nenhum tipo. Nenhuma informação como nome, e-mail, documentos, localização ou dados sensíveis é solicitada ou armazenada.

2. Permissões e Acesso ao Dispositivo

O aplicativo solicita apenas acesso ao armazenamento para salvar o progresso do jogo e arquivos internos. Nenhum arquivo externo do usuário é acessado ou enviado para fora do dispositivo.

3. Dados Técnicos (não pessoais)

O aplicativo pode coletar automaticamente informações não identificáveis, como versão do sistema e modelo do dispositivo, para melhorar estabilidade e desempenho.

4. Crianças

O app não é direcionado especificamente a menores de 13 anos, mas também não coleta informações pessoais independentemente da idade do usuário.

5. Compartilhamento de Dados

  • Não há coleta de dados pessoais.
  • Não há envio de dados para servidores externos.
  • Não há anúncios.
  • Não há integração com redes sociais ou Google Play Games.

6. Segurança

O aplicativo aplica medidas para garantir funcionamento seguro, mesmo sem coletar dados sensíveis.

7. Alterações nesta Política

A Política poderá ser atualizada. A versão mais recente estará sempre disponível neste endereço.
